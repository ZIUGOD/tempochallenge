## Installation

```bash
$ npm install
```

## Database Configuration
  - Create a Postgres Database;
  - Go to src\common\envs\.env and set the database credentials.

## Running the app

```bash
# development
$ npm run start
# or
# watch mode
$ npm run start:dev
# or
# production mode
$ npm run start:prod
```
